## 技术架构和特性

### 前端架构
- **框架**: Next.js 15 + TypeScript
- **样式**: Tailwind CSS v4
- **UI组件**: 自定义组件库（8个核心组件）
- **状态管理**: React hooks
- **响应式设计**: 移动端友好

### 后端架构
- **API**: Next.js API Routes
- **AI集成**: LangChain + OpenAI GPT-4
- **数据处理**: TypeScript + JSON
- **性能优化**: 线性时间复杂度O(n)

### 核心功能特性

#### 1. 智能定位系统
- 多维度用户评分（人口统计学、行为学、心理学）
- 智能渠道推荐（Email vs SMS）
- 紧急程度分析
- 个性化钩子生成

#### 2. AI评分系统
- 四维度消息评分（80-100分制）
- 智能优化建议
- 预测指标计算
- A/B测试变体生成

#### 3. 批量处理系统
- 高性能批量生成（最多100条消息）
- 质量阈值过滤
- 自动优化低分消息
- 多格式导出

#### 4. 实时预览系统
- 毫秒级预览生成
- 一键优化功能
- 实时分数更新
- 状态同步显示
